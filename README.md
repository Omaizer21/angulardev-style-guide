# Angular Best Practices — Folder Structure Guide

A beautiful, interactive, single-page website showcasing the **Angular best practices folder structure** based on the official [Angular Style Guide](https://angular.dev/style-guide).

## 🌐 Live Demo

Visit: [https://omaizer21.github.io](https://omaizer21.github.io)

## ✨ Features

- **Interactive Folder Tree** — Expand/collapse folders, click any item for a detailed explanation
- **Dark / Light Mode** — Toggle with smooth transitions, persisted via localStorage
- **Search & Filter** — Quickly find specific files or folders
- **Copy to Clipboard** — One-click copy of the full folder structure
- **Responsive Design** — Works on mobile, tablet, and desktop
- **Modern UI** — Glassmorphism cards, gradient backgrounds, smooth animations
- **Architecture Overview** — Explains the core / shared / features pattern
- **Naming Conventions** — Reference cards for Angular file types
- **SOLID Principles** — Key design principles for Angular apps
- **CLI Cheat Sheet** — Quick reference for Angular CLI commands

## 🛠️ Tech Stack

- Vanilla HTML, CSS, and JavaScript — no frameworks or build tools
- Google Fonts (Inter + JetBrains Mono)
- Deployed via GitHub Pages

## 📁 Project Structure

```
├── index.html   — Main page with all sections
├── styles.css   — Theming, responsive layout, animations
├── script.js    — Tree rendering, search, theme toggle, interactivity

```
